# 需求文档

## 1. 应用概述

### 1.1 应用名称

雄元科技官网

### 1.2 应用描述

雄元科技官网是中国雄元集团有限公司旗下科技平台的企业门户网站，采用现代科技感设计风格，模仿秒哒官网（miaoda.com）的UI/UX风格，展示企业信息、业务领域、智库平台等内容。网站采用单页响应式设计，支持PC端和移动端访问。

## 2. 用户与使用场景

### 2.1 目标用户

- 潜在客户：寻求环保能源、AI应用、生命科学、高新硬科技等领域解决方案的企业和机构
- 合作伙伴：希望了解雄元科技业务和合作机会的企业
- 求职者：对雄元科技感兴趣的求职人员
- 媒体和公众：关注雄元科技发展动态的社会公众

### 2.2 核心使用场景

- 用户访问官网了解雄元科技的企业背景、业务领域和发展历程
- 用户浏览智库平台，评估企业实力
- 用户通过联系方式或留言表单与雄元科技建立联系

## 4. 业务规则与逻辑

### 4.1 页面滚动与导航

- 点击顶部导航菜单项，页面平滑滚动至对应区域
- 页面滚动时，顶部导航栏背景从透明渐变为半透明深色
- 页面滚动时，触发滚动动画（元素渐入效果）

### 4.2 业务领域卡片交互

- 初始状态：卡片显示业务名称、图标和简要描述
- 点击卡片：展开显示详细内容，其他已展开卡片自动收起
- 再次点击卡片：收起详细内容
- 同一时间只允许一个卡片处于展开状态

### 4.3 留言表单提交

- 用户填写姓名、联系方式、留言内容后，点击提交按钮
- 前端校验必填项，未填写完整时提示「请填写完整信息」
- 提交成功后，显示提示信息「留言已提交，我们会尽快与您联系」
- 提交失败时，提示「提交失败，请稍后重试」

### 4.4 响应式适配

- PC端：导航菜单横向排列，内容区域宽度适配大屏，卡片网格多列展示
- 移动端：导航菜单收起为汉堡菜单，内容区域纵向排列，卡片单列展示，字体和间距适配小屏

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户未填写留言表单必填项 | 提示「请填写完整信息」，禁止提交 |
| 留言表单提交失败 | 提示「提交失败，请稍后重试」 |
| 页面加载失败 | 显示错误提示页面 |
| 图片加载失败 | 显示占位图或默认背景色 |
| 网络断开 | 提示「网络连接失败，请检查网络设置」 |

## 6. 验收标准

1. 用户访问官网，页面加载完成，显示英雄区和顶部导航
2. 用户点击顶部导航「关于我们」，页面平滑滚动至关于我们区域
3. 用户点击业务领域卡片「AI人工智能应用（XYAI）」，卡片展开显示详细内容，其他卡片自动收起
4. 用户滚动至联系我们区域，填写姓名、联系方式、留言内容，点击提交按钮
5. 系统提示「留言已提交，我们会尽快与您联系」
6. 页面在PC端和移动端均正常显示，布局和交互适配良好
