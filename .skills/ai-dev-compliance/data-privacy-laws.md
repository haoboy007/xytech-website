# 数据隐私法规摘要

## 一、中国数据隐私法律体系

### 1.1 核心法律

| 法律 | 生效日期 | 核心要点 |
|------|---------|---------|
| 《网络安全法》 | 2017.6.1 | 网络运营者安全义务、数据本地化、等级保护 |
| 《数据安全法》 | 2021.9.1 | 数据分类分级、重要数据、国家核心数据 |
| 《个人信息保护法》（PIPL） | 2021.11.1 | 中国版"GDPR"，告知-同意、最小必要、敏感信息、跨境规则 |

### 1.2 《个人信息保护法》关键条款

#### 告知-同意（第13-18条）
- **一般规则**：处理个人信息需取得个人**同意**
- **例外情形**：合同必要、法定义务、公共安全、公共卫生、新闻报道等（有限）
- **单独同意**：处理敏感个人信息、向其他处理者提供、公开、跨境传输需单独同意
- **撤回同意**：个人有权撤回，且不得因此拒绝提供服务（处理非必要信息时）

#### 最小必要原则（第6条）
- 不得过度收集
- 处理目的应当明确、合理
- 与处理目的直接相关
- 采取对个人权益影响最小的方式

#### 敏感个人信息（第28-32条）

| 类型 | 示例 | 特殊要求 |
|------|------|---------|
| 生物识别 | 人脸、指纹、虹膜、声纹 | 需特定目的+单独同意+充分必要 |
| 宗教信仰 | 宗教身份、信仰 | 同上 |
| 特定身份 | 身份证号、护照号 | 同上 |
| 医疗健康 | 病历、体检报告 | 同上 |
| 金融账户 | 银行卡号、支付记录 | 同上 |
| 行踪轨迹 | GPS定位、IP地址+时间 | 同上 |
| 未成年人（<14岁） | 全部信息 | 需监护人同意+专人负责 |

#### 个人信息跨境传输（第38-43条）

合法途径（满足其一即可）：
1. 通过国家网信部门安全评估
2. 经专业机构个人信息保护认证
3. 与境外接收方签订标准合同（SCC）
4. 法律法规规定的其他条件

> ⚠️ **关键门槛**：处理超过100万人个人信息的处理者向境外提供个人信息，必须通过国家网信部门安全评估。

### 1.3 App专项合规

#### 《App违法违规收集使用个人信息行为认定方法》

以下行为被认定为违法违规：

**"未公开收集使用规则"**
- 未提供隐私政策
- 隐私政策难以访问（如进入App后超过4次点击才能找到）
- 隐私政策难以阅读（字体过小、颜色过淡）

**"未明示收集使用个人信息的目的、方式和范围"**
- 收集时未逐一列出目的、方式、范围
- 申请打开可收集个人信息权限时未同步告知目的
- 实际收集超出隐私政策所述范围

**"未经用户同意收集使用个人信息"**
- 首次打开即收集（如IMEI、MAC地址）
- 用户明确拒绝后仍收集
- 未经同意向第三方提供
- 未经同意将个人信息用于定向推送

**"违反必要原则，收集与其提供的服务无关的个人信息"**
- 收集与业务功能无关的权限/信息
- 用户拒绝非必要权限后拒绝提供服务
- 申请打开无关权限

**"未经同意向他人提供个人信息"**
- 未经同意嵌入第三方代码（SDK）收集个人信息

**"未按法律规定提供删除或更正个人信息功能"**
- 未提供账号注销功能
- 注销后未删除个人信息或匿名化处理

**"侵犯未成年人在网络空间合法权益"**
- 未经监护人同意收集14岁以下未成年人信息

## 二、欧盟 GDPR

### 2.1 适用范围

- 在欧盟境内设立机构的控制者/处理者
- **向欧盟居民提供商品/服务**（无论机构是否在欧盟）
- **监控欧盟居民行为**（如追踪Cookies）

### 2.2 七大数据处理原则（第5条）

1. **合法、公平、透明**
2. **目的限制**
3. **数据最小化**
4. **准确性**
5. **存储限制**
6. **完整性和保密性**
7. **问责制**

### 2.3 合法基础（第6条）

处理需满足以下至少一项：
1. **同意**（Consent）— 必须自由给予、具体、知情、明确
2. **合同必要**（Contract）
3. **法定义务**（Legal Obligation）
4. ** vital interests**（保护生命利益）
5. **公共利益**（Public Task）
6. **合法利益**（Legitimate Interests）— 需做利益平衡测试

### 2.4 数据主体权利

| 权利 | 内容 | 响应时限 |
|------|------|---------|
| 访问权 | 获取自身数据的副本 | 1个月 |
| 更正权 | 更正不准确数据 | 1个月 |
| 删除权（被遗忘权） | 特定条件下删除数据 | 1个月 |
| 限制处理权 | 暂停处理 | 1个月 |
| 数据可携带权 | 以结构化通用格式获取并转移 | 1个月 |
| 反对权 | 反对基于合法利益/直接营销的处理 | 立即停止 |
| 自动化决策相关权利 | 不受仅基于自动化决策的重大影响 | |

### 2.5 跨境传输机制

| 机制 | 适用条件 | 状态 |
|------|---------|------|
| **充分性认定** | 欧盟认定该国提供充分保护 | 日本、英国、韩国等；美国Privacy Shield被否 |
| **标准合同条款（SCC）** | 与境外接收方签订合同 | 最常用 |
| **约束性企业规则（BCR）** | 跨国集团内部 | 大企业适用 |
| **认证** | 获批准的行为准则/认证 | 较少见 |

> ⚠️ **美国特殊情况**：2023年7月欧盟-美国数据隐私框架（DPF）生效，美国公司可自我认证。但Schrems II案后，仍需进行传输影响评估（TIA）。

## 三、美国 CCPA/CPRA

### 3.1 适用范围

- 在加州开展业务
- 且满足以下任一：年总收入≥$2500万；或购买/接收/共享≥10万消费者/家庭的个人信息；或≥50%收入来自出售个人信息

### 3.2 消费者权利

| 权利 | 内容 |
|------|------|
| 知情权 | 了解收集了哪些个人信息、来源、用途、第三方共享 |
| 访问权 | 获取过去12个月收集的个人信息副本 |
| 删除权 | 要求删除个人信息（有例外） |
| 选择退出权 | 选择不出售/共享个人信息 |
| 非歧视权 | 不因行使权利而被差别对待 |
| 更正权（CPRA新增） | 更正不准确的个人信息 |
| 限制使用敏感信息（CPRA新增） | 限制敏感信息用于某些目的 |

### 3.3 敏感个人信息（CPRA）

包括：SSN、驾照/护照号、地理位置（精确）、种族/民族、宗教、工会成员、基因数据、生物识别信息、健康信息、性取向等。

## 四、合规实操清单

### 4.1 隐私政策必备要素

1. **信息控制者身份**（名称、地址、联系方式）
2. **收集的个人信息类型**（逐一列出）
3. **收集方式**（用户主动提供、自动收集、第三方来源）
4. **使用目的**（每个目的对应具体功能）
5. **共享/转让/委托处理**（接收方身份、目的、类型）
6. **存储期限**（各类信息的保存时间）
7. **用户权利**（访问、更正、删除、撤回同意、注销账号）
8. **跨境传输**（如有，说明接收方国家/地区和保护措施）
9. **Cookies/类似技术**（类型、目的、管理方式）
10. **未成年人保护**（如适用）
11. **联系方式**（隐私问题/投诉渠道）

### 4.2 数据安全措施建议

| 措施 | 必要性 | 实施难度 |
|------|--------|---------|
| 传输加密（TLS 1.2+） | 必须 | 低 |
| 存储加密（AES-256） | 高度建议 | 低 |
| 访问控制（RBAC） | 必须 | 中 |
| 日志审计 | 高度建议 | 中 |
| 定期安全评估 | 建议 | 中 |
| 数据脱敏/匿名化 | 建议 | 中 |
| 备份与恢复 | 必须 | 低 |
| 事件响应计划 | 建议 | 中 |

### 4.3 常见违规处罚案例

| 公司 | 法规 | 处罚 | 原因 |
|------|------|------|------|
| Didi（滴滴） | 《网络安全法》《数据安全法》《PIPL》 | ¥80.26亿元 | 违法违规收集个人信息、数据安全管理不善 |
| Alibaba | 《反垄断法》+ 数据合规 | ¥182.28亿元 | 二选一垄断+数据问题 |
| Meta/Facebook | GDPR | €1.2 billion | 向美国传输数据无充分保护 |
| Amazon | GDPR | €746 million | 定向广告处理合法性 |
| Google | GDPR | €90 million | Cookie同意机制不充分 |

## 五、合规文档模板要点

### 5.1 隐私政策（中国版）

```markdown
隐私政策

【公司/开发者名称】（以下简称"我们"）深知个人信息对您的重要性...

一、我们如何收集和使用您的个人信息
1.1 注册登录：手机号、邮箱、密码
1.2 信息发布：头像、昵称、发布内容
1.3 位置服务：GPS定位（仅在【具体功能】时收集）
...

二、我们如何使用Cookie和同类技术
...

三、我们如何共享、转让、公开披露您的个人信息
...

四、您的权利
您有权：访问、更正、删除您的个人信息；撤回同意；注销账号。
...

五、我们如何保护和存储您的个人信息
存储地点：中华人民共和国境内
存储期限：【具体期限】，超出后将删除或匿名化
...
```

### 5.2 数据处理协议（DPA）要点

与第三方（如云服务、数据分析）共享数据时，应签署DPA，明确：
- 处理目的和范围
- 数据类型和数据主体类别
- 处理者的安全义务
- 子处理者管理
- 数据主体权利协助
- 删除/返还义务
- 审计权
